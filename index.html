功能特点

一个工具，覆盖求职投递全流程——从 Excel 导入到一键发送。

邮件发送

支持 SMTP 直接发送或 IMAP 存草稿审核后再发。两种模式灵活切换，满足不同审慎需求。

批量收件人

上传 Excel 模板一键导入。支持自动解析，覆盖全量收件人信息，批量管理轻松高效。

AI 自动填充

粘贴公司或岗位信息，AI 自动提取公司名、邮箱、主题，再不用手动逐条录入。

变量替换正文

支持 {公司名} {姓名} {学校} 等变量，一封模板自动适配每一家公司。

附件管理

上传简历、求职信等附件，自动附加到每一封邮件。拖拽上传，批量管理。

发送日志

完整发送历史，失败自动标记，支持重试。每次投递都有据可查，不遗漏任何一封。

AI 对话撰写

与 AI 对话定制邮件正文，不满意随时修改。AI 理解语境，生成得体表达。

智能限速

可配置发送速率，避免触发反垃圾机制。默认 20 封/分钟，安心批量投递。

本地安全

所有密钥、配置、数据全部存储在本地。代码完全开源透明，没有隐私泄露风险。

五步投递

从配置邮箱到发送完成，5 步搞定。

1

配置邮箱

选择服务商或手动填写 SMTP/IMAP，测试连接确保可用。

2

导入收件人

上传 Excel 模板或使用 AI 自动提取收件人信息。

3

编辑正文

撰写邮件模板，使用变量自动替换公司名等信息。

4

上传附件

拖拽上传简历和求职信，自动附加到全部邮件。

5

预览发送

逐封预览确认，存草稿或直接发送。全程可控。

开始使用

开源免费，一行命令启动本地服务。

$ git clone https://github.com/777cola/ResumeMailer.git
$ cd ResumeMailer
$ pip install -r requirements.txt
$ python main.py
# → 浏览器打开 http://localhost:8765
